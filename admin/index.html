ℹ️ Copy the code snippet below and paste it into your website's HTML, just before the closing </body> tag.
Basic Embed (Default Config)
<!-- Accessibility Widget --> <script> // Optional: Configure before loading window.AccessibilityWidgetConfig = { lang: 'en', // 'en' | 'ar' | 'auto' position: 'bottom-right', // 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left' primaryColor: '#2563EB' }; </script> <script src="accessibility-widget.js"></script>
Advanced Config Snippet
<!-- Accessibility Widget — Advanced Config --> <script> window.AccessibilityWidgetConfig = { lang: 'auto', position: 'bottom-right', primaryColor: '#2563EB', buttonIcon: '♿', storageKey: 'aw_settings', showProfiles: true, showTTS: true, features: { fontSize: true, fontFamily: true, highContrast: true, darkMode: true, dyslexiaFont: true, readingMask: true, readingGuide: true, textToSpeech: true, keyboardNav: true, // ... set any feature to false to disable } }; </script> <script src="accessibility-widget.js"></script>
Framework Integration
React / Next.js
Import in _app.tsx or use a useEffect to load the script dynamically via script tag injection.
Vue / Nuxt
Add to nuxt.config.js scripts array, or load in app.vue mounted() lifecycle hook.
Angular
Reference in angular.json scripts array or dynamically load from AppComponent ngOnInit.
WordPress
Use wp_enqueue_script() in functions.php or install via the HTML widget in your theme.
Shopify
Add to theme.liquid before tag using the theme editor.
PHP / .NET / Java
Include the script tag in your base layout template file before the closing body tag.
